给定一个数组，将数组中的元素向右移动 k 个位置，其中 k 是非负数。

**示例1：**

```bash
输入: [1,2,3,4,5,6,7] 和 k = 3
输出: [5,6,7,1,2,3,4]
解释:
向右旋转 1 步: [7,1,2,3,4,5,6]
向右旋转 2 步: [6,7,1,2,3,4,5]
向右旋转 3 步: [5,6,7,1,2,3,4]
```
**示例2：**

```bash
输入: [-1,-100,3,99] 和 k = 2
输出: [3,99,-1,-100]
解释: 
向右旋转 1 步: [99,-1,-100,3]
向右旋转 2 步: [3,99,-1,-100]
```

**Solutions：**

```js
/**
 * @param {number[]} nums
 * @param {number} k
 * @return {void} Do not return anything, modify nums in-place instead.
 */
var rotate = function(nums, k) {
  var len = nums.length;  
  var newk = k % len; 
  for(var i = 0; i < newk; i++) {
    var temp = nums[len - 1];
    for(var j = len -2; j >= 0; j--) {  // [1,2,3,4,5,6,7] 
      nums[j+1] = nums[j];
    }
    nums[0] = temp;
  }
};
```
